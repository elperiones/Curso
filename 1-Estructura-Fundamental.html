Bienvenido a Mi Primera Pagina Web

La programación es el arte de transformar ideas lógicas
en soluciones digitales mediante el uso de lenguajes de código.

El desarrollo web se divide principalmente en dos áreas:
el frontend, que es lo que el usuario ve, y el backend, que maneja los datos.

HTML es la estructura fundamental de cualquier sitio web,
permitiendo organizar el contenido de forma semántica y accesible.
